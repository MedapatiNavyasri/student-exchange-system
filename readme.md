# Student Things Exchange System - Landing Page

A premium, modern landing page designed for a university peer-to-peer exchange system. It uses an off-white warm theme with subtle gradients and a classic serif/sans-serif typography combination for high elegance.

## Project Structure

```text
student-exchange-system/
├── index.html        # Main HTML layout, semantic elements
├── style.css         # Styling, utility classes, and CSS custom variables
├── javascript.js     # Interactivity (scroll effects, animations, accordion)
└── README.md         # Project documentation (this file)
```

## Features Demonstrated
*   **Responsive Layout**: Adaptable to Desktop, Tablet, and Mobile views.
*   **Aesthetic Styling**: "Handhold.io" style aesthetic with soft gradients and pill borders.
*   **Intersection Observer**: Elements slide and fade in gracefully on scrolling.
*   **Minimalist Interactive Elements**: Smooth-opening FAQ accordions and interactive buttons.
*   **No Frameworks**: 100% vanilla HTML, CSS, JS.

## How to use
There are no build steps or bundlers required.
1. Open the project folder.
2. Double click the `index.html` file to view it in your browser.
3. Or use a tool like "Live Server" for VSCode.
